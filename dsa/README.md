# DSA - Java Game Source

This folder contains the Java source files for **War of Waves Battle Arena**.

## What is included
- `Main.java` - Entry point for launching the game
- `GameApp.java` - Core game app and scene flow
- `Player.java` - Player state and behavior
- `Enemy.java` - Enemy logic and movement
- `WaveManager.java` - Wave progression and spawn control
- `Gun.java` - Weapon and firing behavior
- `Bullet.java` - Bullet movement and hit handling
- `BloodEffect.java` - Visual damage effect handling

## Tech perspective (Java/OOP)
This part demonstrates Java-based game architecture using object-oriented design:
- Class-based modeling of game entities
- Separation of concerns (player, enemy, weapon, wave management)
- Real-time update logic and interaction between systems
- Foundation for algorithmic thinking in game loops and state transitions
